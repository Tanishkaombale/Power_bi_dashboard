# Data Jobs Dashboard w/ Power BI

![Dashboard page 1](/images/Project1_page1.png)

## Introduction

This dashboard was created for **Job seekers, Job Transitioners , and Job Swappers** to solve a common problems: Information about the data Job market is scattered and hard to grasp. Using a *real-world Dataset of 2024 data science job postings* (including title, salaries and locations), This project provides a single, easy-to-use interface to explore market trends and competition.

## Skills Showcased

- ⚙️ **Data Transformation (ETL) with Power Query:** Cleaned, transformed, and prepared raw job posting data by handling missing values, changing data types, and creating new columns.

- 🧮 **Implicit Measures:** Used Power BI's built-in aggregations such as Average Salary, Median Salary, Job Count, and Maximum Salary to generate quick insights.

- 📊 **Core Charts:** Utilized Column Charts, Bar Charts, Line Charts, and Area Charts to compare job counts, salary trends, and hiring patterns.

- 🌍 **Geospatial Analysis:** Leveraged Map Charts to visualize the geographic distribution of data jobs across different locations.

- 🔢 **KPI Indicators & Tables:** Used Cards and Tables to display important metrics and provide detailed, sortable information.

- 🎨 **Dashboard Design:** Designed an intuitive and visually appealing layout focused on user experience and easy data exploration.

- 🖱️ **Interactive Reporting:**
  - **Slicers:** Enabled dynamic filtering by job title, location, and work type.
  - **Buttons & Bookmarks:** Created a seamless navigation experience between dashboard views.
  - **Drill-Through:** Allowed users to navigate from summary-level insights to detailed job information.

- 📖 **Data Storytelling:** Combined multiple visualizations to communicate job market trends, salary insights, and skill demand effectively.
